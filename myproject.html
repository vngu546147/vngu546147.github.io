My Project JS

addNavItemClickListeners:

The JavaScript code was added with a function called addNavItemClickListeners which is used to add click event listeners to navigation items. It selects the navigation element and all the list items within it, then iterates through each list item and adds a click event listener to it. When a list item is clicked, it removes the 'active' class from all list items and adds the 'active' class to the clicked list item, allowing for the navigation items to be highlighted when clicked, indicating which page the user is currently on. This is provide in myproject, picture-gallery, info, index, calculator and number-game-errors

Calculator:

The JavaScript function calc() it performs basic arithmetic operations on input values based on a selected operator. It retrieves input values and performs calculations according to the chosen operator. If division or modulus is performed, it checks if the second value is zero to avoid errors. The result is displayed in an HTML element. If an error occurs, it alerts and returns without further calculations.Sample calculator

Validation:

The code uses JavaScript to validate user input for email and phone number fields. It adds event listeners to the input fields, triggering validateEmail and validatePhone functions. The validateEmail function checks if the email address matches a valid pattern, while the validatePhone function checks if the phone number matches a valid pattern.
